# Task Manager Desktop Client

## Что делает клиент
Windows Forms приложение на C# для работы с системой управления задачами. Позволяет пользователям регистрироваться, создавать проекты и задачи, добавлять комментарии через графический интерфейс.

## С каким API работает
Работает с Task Manager API (https://github.com/MariaKiiseleva/task-tracker-lab1/tree/lab2). API предоставляет RESTful интерфейс для управления пользователями, проектами, задачами и комментариями. Использует JWT-аутентификацию и поддерживает идемпотентные операции.

## Примеры использования
- **Регистрация**: POST /api/v1/auth/register
- **Создание проекта**: POST /api/v1/projects/ с Idempotency-Key
- **Создание задачи**: POST /api/v1/projects/{id}/tasks
- **Просмотр комментариев**: GET /api/v1/tasks/{id}/comments

## Форматы ответов
- **Успешная аутентификация**: `{"access_token": "jwt_token", "token_type": "bearer"}`
- **Проект**: `{"id": 1, "name": "Название", "description": "Описание", "owner_id": 1}`
- **Задача**: `{"id": 1, "title": "Задача", "description": "Описание", "project_id": 1, "status": "open", "priority": 3}`

Клиент реализует все базовые CRUD операции, пагинацию и обработку ошибок. Подходит для изучения интеграции с REST API на практике.
