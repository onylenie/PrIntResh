# Опциональные поля ответа (include)

API поддерживает механизм выборочного расширения ответа с помощью
query-параметра:

?include=поле1,поле2

Это позволяет клиенту получать дополнительные вложенные данные
по необходимости, без лишних запросов.

## Доступные include-поля

Для задач (v1 и v2):

include=project
include=comments
include=project,comments

## Зачем это нужно

- уменьшение количества запросов  
- гибкость клиентского приложения  
- точечная загрузка только нужных данных  
- улучшение производительности
